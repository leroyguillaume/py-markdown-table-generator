# py-markdown-table-generator <!-- omit in toc -->

Python library to generate Markdown table.

- [Build](#build)

## Build

```bash
python3 -m venv venv
. ./venv/bin/activate
pip install -r requirements.txt
python3 -m build
```

## Test

```bash
python3 -m venv venv
. ./venv/bin/activate
pip install -r requirements.txt
pytest
```
